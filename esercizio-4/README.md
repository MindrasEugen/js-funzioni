# esercizio

Un taxi applica una tariffa extra fissa dopo la mezzanotte. Scrivi una funzione che calcoli il totale della corsa dato il prezzo base, l'orario di partenza e l'orario di arrivo.

## ragionamento 

- definisco prezzo base
- definisco l'orario di partenza
- definisco l'orario di arrivo
- definiamo tariffa extra = 5 €
- cacola la tariffa
- stampa il prezzo
