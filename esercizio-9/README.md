# esercizio

Etichetta l'utente in base alla sua età. Sotto i 18 anni "Minorenne", "Giovane Adulto" tra 18 e 35, "Adulto" se tra 36 e 60, "Senior" oltre 60.

## ragionamento

- Chiedo al utente di inserire la sua età
- SE età < 18
    - stampo "Minorenne"
- ALTRIMENTI 
    - SE età < 35 
        - stampo "Giovane Adulto"
- ALTRIMENTI
    - SE età < 60
        - stampo "Adulto"
- ALTRIMENTI
    - stampo "Senior"
